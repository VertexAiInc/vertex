## 🌟 About Vertex

**Vertex AI** is an advanced conversational artificial intelligence built on **Llama 3.3** and accelerated by **Groq's ultra-fast inference engine**.

Designed to deliver natural conversations, rapid responses and a clean modern interface, Vertex combines cutting-edge AI performance with an intuitive user experience.
