# Security Policy

## Supported Versions

Apenas a versão mais recente do Vertex AI recebe atualizações de segurança.

| Version | Supported |
| ------- | :-------: |
| 1.6     |     ✅     |
| < 1.6   |     ❌     |

---

## Reporting a Vulnerability

A segurança do Vertex AI é importante para nós.

Se você encontrar uma vulnerabilidade de segurança, **não a divulgue publicamente** antes que possamos analisá-la.

Em vez disso, entre em contato com a equipe da VertexAiInc informando:

* Uma descrição clara da vulnerabilidade.
* Passos para reproduzir o problema.
* O impacto esperado.
* Capturas de tela ou vídeos (se necessário).

Após recebermos o relatório, iremos:

1. Confirmar o recebimento.
2. Investigar o problema.
3. Desenvolver uma correção, quando aplicável.
4. Publicar a atualização em uma nova versão.

---

## Scope

Esta política se aplica aos seguintes projetos da VertexAiInc:

* Vertex AI Web
* Vertex AI Android
* Repositórios oficiais da VertexAiInc

---

## Responsible Disclosure

Pedimos que qualquer vulnerabilidade seja reportada de forma responsável.

Evite:

* Publicar exploits antes da correção.
* Compartilhar informações que possam colocar usuários em risco.
* Explorar vulnerabilidades para obter acesso não autorizado.

---

## Security Updates

Correções de segurança serão disponibilizadas na versão mais recente do projeto.

Versões antigas podem não receber atualizações.

---

## Contact

Caso deseje reportar uma vulnerabilidade, utilize os canais oficiais da VertexAiInc disponíveis em nossos repositórios do GitHub.
